# AI-ML-PROJECTS

A collection of Python-based AI/ML projects demonstrating key machine learning techniques.

## Projects

1. **Customer Churn Prediction**  
   Predict if a customer will leave a service using classification models.  
   `customer-churn-prediction/customer_churn_prediction.py`

2. **House Price Prediction**  
   Predict house prices using regression models.  
   `house-price-prediction/house_price_prediction.py`

3. **Spam Email Detection**  
   Classify emails as spam or not using text classification.  
   `spam-email-detection/spam_detection.py`
